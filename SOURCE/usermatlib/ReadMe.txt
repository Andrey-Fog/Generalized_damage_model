Обозначения 

σ - тензор напряжений (6 компонент)
G - модуль Сдвига
σ_y0 - начальный предел текучести
ε_pl -тензор пластических деформаций (6 компонент)
ε_cr - тензор деформаций ползучести (6 компонент)
ε_cr_eqv = √(2/3 ε_cr:ε_cr)- эквивалентные деформации ползучести, скаляр
ε_pl_eqv = √(2/3 ε_pl:ε_pl)  - эквивалентные пластические деформации, скаляр
X = X1 + X2 + X3 - сумма тензоров девиаторов остаточных (обратных) напряжений (6 компонент)
R - радиус поверхности текучести, скаляр
λ - пластический множитель, скаляр 
D = D_pl +  D_cr - D_cr*D_pl- общая мера поврежденности, скаляр
D_pl - поврежденность при пластичности, скаляр
D_cr - поврежденность при ползучести, скаляр
σ_eff = σ / (1 - D)- тензор эффективных напряжений, (6 компонент)
s_eff = σ_eff - (1/3)·tr(σ_eff)·I - тензор девиатор эффективных напряжений, (6 компонент)
σ_eff_H = ((σ_eff_11 + σ_eff_22+ σ_eff_33) / 3) - гидростатические напряжения, скаляр 
σ_eff_eqv = √{(1/2)·[(σ_eff_11-σ_eff_22)² + (σ_eff_22-σ_eff_33)² + (σ_eff_33-σ_eff_11)² + 6·(σ_eff_12²+σ_eff_23²+σ_eff_13²)]}
С - тензор упругости 4-го ранга
Y = (σ_eff_eqv^2) / (2E(1-D_pl)^2) * [2/3*(1+ν) + 3(1-2ν) * (σ_eff_H/σ_eff_eqv)^2 ], скаляр

R = A_iso *  (ε_pl_eqv)^B_iso - радиус поверхности текучести


System of equations
Уравнение 1: Упругость в девиаторах (тензорное, 6 компомнет)
Φ_elastic = σ_eff - C : (ε_total - ε_pl - ε_cr) = 0
Уравнение 2: Поток пластичности (тезорное, 6 компонент)
Φ_flow = ε_pl - ε_pl_0 - depseqv * (3/2) * (s_eff - X) / √(3/2 (s_eff - X):(s_eff - X)) = 0
Уравнение 3: Интегрирование ползучести (Modified Norton, тензорное, 6 компонент)
Φ_cr = ε_cr - ε_cr_0 - Δt * (3/2) * A_cr * (σ_eff_eqv)^(B_cr-1) * s_eff = 0
Уравнение 4: Интегрирование кинематического упрочнения (Сhaboche, тензорное, 6 компомнет)
Φ_X1 = X1 - X1_0 - ((2/3) * C1_chb * (ε_pl - ε_pl_0) - Y1_chb * X1 * (ε_pl_eqv - ε_pl_eqv_0)) = 0
Уравнение 5: Интегрирование кинематического упрочнения (Сhaboche, тензорное, 6 компомнет)
Φ_X2 = X2 - X2_0 - ((2/3) * C2_chb * (ε_pl - ε_pl_0) - Y2_chb * X2 * (ε_pl_eqv - ε_pl_eqv_0)) = 0
Уравнение 6: Интегрирование кинематического упрочнения (Сhaboche, тензорное, 6 компомнет)
Φ_X3 = X3 - X3_0 - ((2/3) * C3_chb * (ε_pl - ε_pl_0) - Y3_chb * X3 * (ε_pl_eqv - ε_pl_eqv_0)) = 0
Уравнение 7: Условие пластичности (скалярное, 1 компонента)
Φ_pl = √(3/2 (s_eff - X):(s_eff - X)) - [σ_y0 + A_iso * (ε_pl_eqv)^B_iso] = 0
Уравнение 8: Интегрирование пластического повреждения (Lemaitre, , скалярное, 1 компонента)
 Φ_Dpl = D_pl - D_pl_0 - Δt * (Y/A_lem)^B_lem * (Depseqv/Δt) * (1 - D)^(-C_lem) = 0
Уравнение 9: Интегрирование повреждения ползучести (Rabotnov-Kachanov, скалярное, 1 компонента)
Φ_Dcr = D_cr - D_cr_0 - Δt * A_dmg * (σ_eff_eqv)^B_dmg


σ_eff_eqv = √{(1/2) · [(σ_eff_11 - σ_eff_22)^2 + (σ_eff_22 - σ_eff_33)^2 + (σ_eff_33 - σ_eff_11)^2 + 6·σ_eff_12^2 + 6·σ_eff_23^2 + 6·σ_eff_13^2]}
Напряжения
σ = (1 - D) * C : (ε_total - ε_pl - ε_cr)
