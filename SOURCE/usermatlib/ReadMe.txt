System of equations

Уравнение 1: Упругость
σ = (1 - D) * C : (ε_total - ε_pl - ε_cr)
Уравнение 2: Условие пластичности
Φ_pl = √(3/2 (s_eff - X):(s_eff - X)) - [σ_y0 + R] = 0
Уравнение 3: Интегрирование ползучести (Модифицированный Нортон)
Φ_cr = ε_cr - ε_cr₀ - Δt * (3/2) * A * (σ_eff)^(n-1) * s * (ε_cr_eqv)^q = 0
Уравнение 4: Интегрирование эквивалентной деформации ползучести (Модифицированный Нортон)
Φ_cr_eqv = ε_cr_eqv - ε_cr_eqv₀ - Δt * A * (σ_eff)^n * (ε_cr_eqv)^q = 0
Уравнение 5: Интегрирование пластического повреждения
Φ_Dpl = D_pl - D_pl₀ - Δt * (Y/S)^s * (Δε_pl_eqv/Δt) * (1 - D)^(-k) = 0
Уравнение 6: Интегрирование повреждения ползучести
Φ_Dcr = D_cr - D_cr₀ - Δt * A_cr * (σ_eff)^r = 0
Уравнение 7: Интегрирование кинематического упрочнения (Сhaboche)
Φ_X1 = X1 - X1₀ - (2/3) * C1_X * (ε_pl - ε_pl₀) + γ1_X * X * (ε_pl_eqv - ε_pl_eqv₀) = 0
Уравнение 8: Интегрирование кинематического упрочнения (Сhaboche)
Φ_X2 = X2 - X2₀ - (2/3) * C2_X * (ε_pl - ε_pl₀) + γ2_X * X * (ε_pl_eqv - ε_pl_eqv₀) = 0
Уравнение 9: Интегрирование кинематического упрочнения (Сhaboche)
Φ_X3 = X3 - X3₀ - (2/3) * C3_X * (ε_pl - ε_pl₀) + γ3_X * X * (ε_pl_eqv - ε_pl_eqv₀) = 0
Уравнение 10: Интегрирование изотропного упрочнения (Power law)
Φ_R = R - K * (ε_pl_eqv)^m = 0
Уравнение 11: Определение пластической деформации
Φ_flow = ε_pl - ε_pl₀ - Δλ * (3/2) * (s_eff - X) / √(3/2 (s_eff - X):(s_eff - X)) = 0
Уравнение 12: Связь параметра упрочнения
Φ_alpha = ε_pl_eqv - ε_pl_eqv₀ - Δλ = 0

В этой системе 

σ - тензор напряжений
ε_pl -тензор пластических деформаций
ε_cr - тензор деформаций ползучести
ε_cr_eqv - эквивалентные деформации ползучести, скаляр
ε_pl_eqv - эквивалентные пластические деформации, скаляр
X = X1 + X2 + X3 - суммарный тензор остаточных (обратных) напряжений
R - радиус поверхности текучести, скаляр
λ - пластический множитель, скаляр 
X - тензор девиатор остаточных напряжений
D=D_pl +  D_cr - D_cr*D_pl- общая мера поврежденности, скаляр
D_pl - поврежденность при пластичности, скаляр
D_cr - поврежденность при ползучести, скаляр
σ_eff = σ_eff / (1 - D)- тензор эффективных напряжений
s_eff = s / (1 - D) тензор девиатор эффективных напряжений
С - тензор упругости 4-го ранга
Y = (σ_eff^2) / (2E(1-D_pl)^2) * [2/3*(1+ν) + 3(1-2ν) * (σ_H/σ_eff)^2 ]
σ_H - гидростатические напряжения
